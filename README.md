## Goal
Using react to visualize the COVID-19 Tests given, and their results.

## Data Source
Data was collected using the covid tracker api for states in the US.  

_Projects was inspired by the work of Justin Juno_.  

## Insights
From the data, it is worth mentioning that some records were incomplete, Looking at the values of the total tests given and the results of positives and negatives.
